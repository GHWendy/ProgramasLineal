-En la carpeta se encuentran cuatro archivos, un archivo con extensión
 ".c" es donde se encuentra todo el código que conllevó el ejercicio,
por otra parte, hay un ejecutable llamado "Elim-GaussEje", este cuando se 
ejecuta realiza la operación (lee el archivo input1.txt y da una salida
outpu1.txt).

necesita estar dentro del mismo directorio, tanto del codigo y del ejecutable.

satisfactoriamente.

-Se puede modificar los valores que se encuentrar dentro del archivo input1.txt
